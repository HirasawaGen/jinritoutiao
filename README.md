## 项目说明

本项目基于python3.12开发，依赖管理工具使用uv，主要的依赖有playwright aiohttp beautifulsoup4 等

项目主要功能为：从今日头条上批量下载视频，并批量上传到今日头条。

## 依赖说明

- playwright 用于自动化操作浏览器，实现自动化下载视频与上传

- aiohttp 用于异步网络请求，实现批量下载视频

- beautifulsoup4 用于解析网页

- aiosqlite 用于数据库操作，后面可能会改成aiomysql

为了让用户不要敲生冷的命令，而是可以图形化操作，项目后期可能会使用fastapi+uvicorn制作webui界面。

## 主要功能的实现情况

- 批量下载视频：实现了一半。今日头条上有的视频在服务器里就是一个视频文件，这些很好做。但是有些视频是画面和音频分离的，需要用到ffmpeg等工具才能合并成一个文件。其实也不难做，后面在做，我先把难做的解决了。

- 自动登录：同样实现了一半。不过会将账号的cookies保存到本地数据库，所以除非首次登录或者cookies过期，否则不需要再次登录。

- 批量上传视频：未实现

- 视频去水印：未实现

- 视频去标题党：未实现

- 图形化界面：未实现
